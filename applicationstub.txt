The purpose of this application is to form a robust application that standardize
the the application development processing and show best practices of 
application design in the c programming language.

Design considerations:
1) Error handling
2) Signal handling
3) Performance issues
4) Logging of code, errors, and events within the application.
5) Memory Leaks
6) Application security from known flaws within the operating system and IPC
7) Communication from machine to machine 
8) Communication from processor to processor
